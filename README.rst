Install
------------------------------------------------------------------------------

``boto3_dataclass`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install boto3-dataclass

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade boto3-dataclass
